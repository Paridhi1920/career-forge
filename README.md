# CareerForge 🚀

CareerForge is an AI-powered Resume and Cover Letter Generator built using Streamlit and Hugging Face models.

## Features
- ATS-friendly resume generation
- Custom cover letter creation
- Role-based content generation
- PDF resume export
- Clean and simple UI

## Tech Stack
- Python
- Streamlit
- Hugging Face Inference API (flan-t5-base)
- FPDF
- dotenv

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
